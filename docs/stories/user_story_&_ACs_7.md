## User Story: Budgeting and Goal Setting
As a user, I want to seet financial goals and receive budgeting recommendations so that I can manage my money effectively.

## Acceptance Criteria
- **AC1:** Given that I enter my monthly income, When the system process it, Then it should calculate a suggested budget.
- **AC2:** Given that spending data exists, When analyzed, Then the system should recommend budget allocations per category
- **AC3:** Given that I define priority categories, When budgeting is calculated, Then recommendations should reflect my priorities.
- **AC4:** Given that spending exceeds a category, When analyzed, Then the system should flag it.
- **AC5:** Given that recommendations are show, When displayed, Then they should be clear and easy to understand
