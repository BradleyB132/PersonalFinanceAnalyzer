## User Story: Download Reports
As a user, I want to download reports of my financial data so that I can save or share them.

## Acceptance Criteria
- **AC1:** Given that I have transaction data, When I click export, then I should be able to download a file
- **AC2:** Given that I choose PDF export, When generated, Then I should receive a snapshot of my dashboard
- **AC3:** Given that export is successful, When download completes, Then the file should contain accurate data.
