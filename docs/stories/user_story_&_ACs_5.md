### User Story: Search and Filter Transactions
As a user, I want to serach and filter my transactions so that I can find specific data quickly.

### Acceptance Criteria

- **AC1:** Given that transactions exist, When I search by keyword, Then matching results should be displayed.
- **AC2:** Given that filters are applied, When I select date range, category, or amount, then results should update.
- **AC3:** Given that no resluts match, When a search is performed, Then a no "no results" message should appear.
