### User Story: View Dashboard

As a user, I want to view a dashboard of my financial data so that I can understand my spending patterns

### Acceptance Criteria

- **AC1:** Given that transaction are uplaoded, when processsing completes, Then each transaction should have a category.
- **AC2:** Given that transaction data exists, When I view the dashboard, Then I should see a chart of spending by category
- **AC3:** Given that transaction data exists, When I view the dashboard, Then I should see spending trends over time.
- **AC4:** Given that no transactions are uploaded, When I open the dashboard, Then I should have a button to upload data
